BUILD REQUIRED

dist/main.js not found

You cloned the repo — now you need to build the bundle.

1) Install deps (one time):
       npm install

2) Install Rust WASM toolchain (one time, only if not already installed):
       rustup target add wasm32-unknown-unknown
       cargo install wasm-bindgen-cli --version 0.2.95

3) Build:
       ./build.sh

4) Serve and play:
       python3 -m http.server 8000
       (then open http://localhost:8000)

If you already built, make sure you are serving the
project root (not a subfolder) so dist/main.js resolves.

Once built, refresh this page.
